HOW TO USE (3 steps)
--------------------
1. Open  saas-landing.html  in your editor (or just double-click to preview in a browser).
2. Replace every {{PLACEHOLDER}} with your own words. Change the accent color by
   find-replacing "indigo" with any Tailwind color (e.g. "emerald", "rose").
3. Deploy free: drag the file into Netlify Drop (app.netlify.com/drop), or push to
   Vercel / GitHub Pages. Live in minutes. No framework, no build step.
